# PixEngine Examples

실제 작동하는 예제 모음입니다.

## 설치

```bash
cd examples
pnpm install
```

## 실행

```bash

# 기본 사용법

pnpm basic

# 여러 포맷 생성

pnpm formats

# 동적 정책

pnpm dynamic

# 모든 예제 실행

pnpm all
```

## 예제 설명

- **basic.ts**: 기본 사용법 - 2개의 WebP variant 생성
- **multiple-formats.ts**: WebP, AVIF, JPEG 다양한 포맷 생성
- **dynamic-policy.ts**: 이미지 크기에 따라 동적으로 전략 변경
